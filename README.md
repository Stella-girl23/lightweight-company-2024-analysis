Lightweight Company 2024 Operations Analysis
Overview
This repository contains a detailed data analysis project on the 2024 operations of Lightweight Company, a solar panel sales, installation, and renewable-energy services business.
The purpose of this project is to document the full workflow of a real-world data analysis process—from data generation to cleaning, analysis, visualization, and insights using Excel and Power BI.

Objectives of This Project
To analyse 10,000 operational records from Lightweight Company.
To clean and structure raw data using Microsoft Excel.
To build interactive dashboards using Power BI.
To derive actionable insights that inform decision-making.
To document the full analytical process on GitHub for portfolio purposes.

Tools Used
Microsoft Excel – data cleaning & preparation
Power BI – visualization & business intelligence
GitHub – documentation, version control, portfolio
CSV Dataset – 10,000 generated operational records
